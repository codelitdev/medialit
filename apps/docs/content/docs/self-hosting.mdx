## Bucket Policy
```json

    "Version": "2012-10-17",

        {
            "Sid": "PublicReadAccess",
            "Effect": "Allow",
            "Principal": "*",
            "Action": "s3:GetObject",
            "Resource": "arn:aws:s3:::medialit-temp-refactoring/medialit-service/public/*"
        }
    ]
}
```
